Computes attitude of a GNSS satellite

This function calculates the satellite body reference frame based on position, velocity and sun position vectors according to IS-GPS-200H and similar standards.

参数
[in]satindexIndex of the satellite in the constellation, the satellite index obtained by function satsysprn2index() in gnss_config.c
[in]satposSatellite position vector in ECEF [m] (input)
[in]satvelSatellite velocity vector in ECEF [m/s] (input)
[in]sunposSun position vector in ECEF [m] (input)
[out]exUnit vector in X direction of satellite body frame (output)
[out]eyUnit vector in Y direction of satellite body frame (output)
[out]ezUnit vector in Z direction of satellite body frame (output)
[out]epUnit vector pointing to the sun projected onto the spacecraft y-z plane (output)
注解
The body frame follows the convention:
  • Z-axis (ez) points to the Earth center (nadir)
  • Y-axis (ey) is perpendicular to both Z-axis and sun direction
  • X-axis (ex) completes the right-handed system satpos and sunpos have to be in the same frame and in the same unit.!!

int read_att_quaternion_corbit ( char *  quaternion_att_filename,
ATT_QUATERNION *  qua_store,
int  record_start 
)

Reads attitude quaternion data from a file(corbit defined format) and stores it in the provided structure.

This function reads quaternion attitude data from a specified file and stores the data starting from a given record index into the provided ATT_QUATERNION structure.

参数
[in]quaternion_att_filenameThe name of the file containing the quaternion attitude data.
[in,out]qua_storeA pointer to an ATT_QUATERNION structure where the read data will be stored.
[in]record_startThe index of the record from which to start reading the data.
返回
the number of quaternion read from this file
注解
file format:

mjd[%16.9lf] q0[%16.14lf] q1[%16.14lf] q2[%16.14lf] q3[%16.14lf]
