◆ filter_()

int filter_ ( double *  x,
double *  P,
double *  H,
double *  v,
double *  R,
int  n,
int  m,
double *  xp,
double *  Pp 
)

this is the implementation of kalman filter

注解
the formulae for kalman fileter K = P*H'*(H*P*H'+R)^-1 xp = x + K*v Pp = (I - K*H)*P
参数
[in]xstates vector (n by 1)
[in]Pcovariance matrix of states (n by n)
[in]Hdesign matrix (m by n)
[in]vinnovation (measurement - model) m by 1
[in]Rcovariance matrix of measurement err (m by m)
[in]nnumber of states
[in]mnumber of measurements
[out]xpstates vector after update (n by 1)
[out]Ppcovariance of the states after update (n by n)
返回
status (0:ok , <0: error)

< H*P = HP

< HP*H' = HPHT

< P*H' = PHT, here we use HP to represent PHT

< K = P*H'*Q^-1

< K*v = Kv

KH = I - K*H

Pp = (I - K*H)*P
