compute the matrix from ecef xyz to ENU coordinates https://gssc.esa.int/navipedia/index.php/Transformations_between_ECEF_and_ENU_coordinates

参数

[out]Ethe convertion matrix,
注解

\[ \mathbf{X_{enu}} = \mathbf{E} \cdot \mathbf{Y_{ecef}} \]

\[ \mathbf{\hat{e}}=(E[0],E[1],E[2])^T \newline \mathbf{\hat{n}}=(E[3],E[4],E[5])^T \newline \mathbf{\hat{u}}=(E[6],E[7],E[8])^T \newline \]
