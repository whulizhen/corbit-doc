void azel ( double *  blh,
double *  dxyz,
double *  azel 
)

Calculate azimuth and elevation from user given coordinates. compute the azimuth and elevation for ground users, this is not for the platform coordiante, but for the ground ENU cooridnates. This function computes the azimuth and elevation angles based on the provided geodetic coordinates (latitude, longitude, height) and the difference in Cartesian coordinates.

参数
[in]blhPointer to an array containing the geodetic coordinates:
  • blh[0]: Latitude in radians
  • blh[1]: Longitude in radians
  • blh[2]: Height in meters
[in]dxyzPointer to an array containing the differences in ECEF Cartesian coordinates:
  • dxyz[0]: Difference in X coordinate in meters
  • dxyz[1]: Difference in Y coordinate in meters
  • dxyz[2]: Difference in Z coordinate in meters
[out]azelPointer to an array where the calculated azimuth and elevation will be stored:
  • azel[0]: Azimuth in radians
  • azel[1]: Elevation in radians

Computes the azimuth and elevation angles of a signal.

◆ xyz2enu_matrix()

void xyz2enu_matrix ( double *  blh,
double *  E 
)

Converts geodetic coordinates (latitude, longitude, height) to an ENU (East-North-Up) transformation matrix.

This function takes geodetic coordinates (latitude, longitude, height) and computes the corresponding transformation matrix to convert from ECEF (Earth-Centered, Earth-Fixed) coordinates to ENU (East-North-Up) coordinates.

参数
[in]blhPointer to an array containing the geodetic coordinates: blh[0] - Latitude in radians blh[1] - Longitude in radians blh[2] - Height in meters
[out]EPointer to a 3x3 array where the resulting ENU transformation matrix will be stored. https://gssc.esa.int/navipedia/index.php/Transformations_between_ECEF_and_ENU_coordinates
注解

\[ \mathbf{X_{enu}} = \mathbf{E} \cdot \mathbf{Y_{ecef}} \]

\[ \mathbf{\hat{e}}=(E[0],E[1],E[2])^T \newline \mathbf{\hat{n}}=(E[3],E[4],E[5])^T \newline \mathbf{\hat{u}}=(E[6],E[7],E[8])^T \newline \]
