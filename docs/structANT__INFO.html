Detailed Description

Antenna information for both the GNSS satellite and the receiver antenna.

Note
for gnss satellite antenna: ant_code is G01, E01,... ant_type is the block type of the gnss satellite for ground receiver antenna: ant_code are all spaces, ant_type is the type of the ground antenna (might contain empty spaces), like "SEPVC6150L SCIS". idx_object is the satindex for gnss satellites and idx_object=-1 for ground receivers installation matrix is from antenna frame to the platform frame
