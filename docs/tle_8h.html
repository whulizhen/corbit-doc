Searches for a TLE (Two-Line Element) entry in the TLE database.

This function searches for a TLE entry in the provided TLE database that matches the given satellite number and epoch time.

参数
[in]tle_databasePointer to the TLE database array.
[in]satnoThe satellite number to search for.
[in]epoch_utcThe epoch time in UTC to match.
返回
Index of the matching TLE entry in the database, or -1 if not found.
