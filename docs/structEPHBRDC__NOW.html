详细描述

struct EPHBRDC_NOW is a temperay variable to store the choosed brdc pointers for current time epoch.

注解
all the members are pointers to EPHBRDC_STORE, therefore, there is no need to allocate memory for pointers in EPHBRDC_NOW. The memory for a pointer is 8 bytes
