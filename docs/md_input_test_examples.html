How to use COrbit library?

You will need a Ubuntu Linux system (22.04.1 LTS) or the windows WSL (with ubuntu 22.04.1 LTS) installed. The static library "libcorbit.a" and dynamic library "libcorbit.so" have to be compiled with the header files int the "include" directory. The following is an example of using libcorbit with the gcc complier.

If you have a source file named test.c. In this file, you write the following code.

#include <stdlib.h>
#include <stdio.h>
#include "gtime.h"
#include "common_function.h"
#include "mathlib.h"
#include "orbit_dynamics.h"
#include "keplerian_element.h"
#include "gnss_satposition.h"
#include "tle.h"
#include "gnss_config.h"
//here is a function of computing GPS satellite position using broadcast ephemeris
void test_brdc_gps()
{

The command is as follows:

gcc test.c -std=c99 -fPIC -O3 -I ${your_libcorbit_include} -L ${your_libcorbit_directory} -lcorbit -lm -o test -D ULOG_ENABLED

Among which, ${your_libcorbit_include} is path for all the header files. ${your_libcorbit_directory} is path for the "libcoribt.so" and "libcorbit.a". Once you press Enter, it will compile and produce a program named "test". You can run the prorame "test" by commond "./test". Then you will see the ecef position of a GPS satellite on the screen.
