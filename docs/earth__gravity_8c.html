◆ computeNormalFactor()

void computeNormalFactor ( )

calculate the normalization facor for C and S, eq. 3.16 in page 58 in Montenbruck Nnm = sqrt( sqrt(2-d0m)*(2n+1)*(n-m)!/(n+m)! ) Pnm_normalized = Nnm * Pnm ; but, Cnm = Nnm* Cnm_normalized; Snm = Nnm* Snm_normalized;

注解
recursive formulae: Fn,m = Fn-1,m * sqrt( (n-m)(2n+1)/(n+m)/(2n-1) ) Fm,m = Fm-1,m-1 *sqrt( 2m+1/(2m-1)/(2m)/(2m-1) )

compute ocean tide correction due to the denormalized earth geopotential coefficient IERS technote 2010, the FES2014b ocean tide model in spherical harmonics in geopotential units are downloaded from https://grace.obs-mip.fr/dealiasing_and_tides/ocean-tides/ IERS technote 2010, chapter6, eq. 6.15

注解
Note that, for zonal terms, FES2004 takes the approach to set the retrograde coe cients C-fn0 and S-fn0 to zero and to double the prograde coefficients C+fn0 and S+fn0. Therefore, after applying Equation (6.15), the Cn0 have the expected value but the Sn0 must be set to zero. For FES2014b, is it necessary to set Cn0 ZERO???
