int eph_header_jpl::ipt[TOTAL_PLANET][3]

Word (1,i) is the starting location in each data record of the chebychev coefficients belonging to the ith item.

Word (2,i) is the number of chebychev coefficients per component of the ith item, Word (3,i) is the number of complete sets of coefficients in each data record for the ith item.
