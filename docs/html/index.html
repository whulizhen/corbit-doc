Overview

COrbit is a lightweight distributed orbit analysis tool in standard C language. It supports GNSS data processing include pseudorange positioning, carrier phase based pricise positioning, Real-Time Kinematics positioning and very precise orbit dynamics.

Features

  • lightweight but very precise orbit dynamics, includeing high fidelity non-conservative force modelling
  • simplified but accurate earth gravitational acceleraiton model, designed for onboard computation
  • orbit prediction with numerical integration
  • gnss data processing includes spp, ppp and rtk
  • satellite laser ranging data processing including orbit residual check, orbit determination and total solar irradiance estimation.
  • support realtime data processing, suitable for LEO satellite onborad realtime orbit determination

System requirements

In order to compile the libcorbit, the build essentials must be installed.

  • Ubuntu 22.04.1 LTS or the WSL( Windows Subsystem for Linux)
  • gcc 11.3.0
  • GNU Make 4.3 if you use the Make file [optional]
  • Python 3.9 for generating the pictures of results [optional]
