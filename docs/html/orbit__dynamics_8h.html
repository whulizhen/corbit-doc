Returns
void

the orbit derivatives function

Parameters
[in]nstatedimension of the orbit state vector
[in]sec_since_startseconds since the start of orbit integration
[in]ythe orbit state vector, including pos, vel, PHI, and S
[out]dydxthe derivatives of orbit state, includeing vel ,acc, dPHi, and dS
Returns
void
Note
here y = (x,y,z,u,v,w,Phi,S) dydt = (u,v,w,ax,ay,az,dPhi/dt,dS/dt)

typedef void (Derivative)(int,double,double, double* ); compute_acc_all has to be the same as Derivative function pointer it compute the dydx at time mjd_utc based on the nstate dimentional y.

y=(x,y,z,u,v,w)T; f=(u,v,w,ax,ay,az); dy/dt = (u,v,w,ax,ay,az)T;

y = (y;Phi) here y = (x,y,z,u,v,w,Phi,S) dydt = (u,v,w,ax,ay,az,dPhi/dt,dS/dt)

deal with dadr and dadp for orbit determination to compute dPHI/dt and dS/dt Note that, y = (r,v), f=(v,a) dPHI/dt = df/dy * PHI PHI = dy/dy0 ; PHI(t0) = I df/dy is like this :

0 I
da/dr da/dv

assume da/dv = 0

if considering the sensitivity matrix S: S = dy/dp ; S(t0) = 0 

                   |  0    |

dS/dt = df/dy * S + | | | da/dp |, 6 by np

dPHI/dt = df/dy*PHI 

                            |0   0     |

d(PHI,S)/dt = dF/dy * (PHI,S) + | | |0 da/dp |, 6 by (6+np)
