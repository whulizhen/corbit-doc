solve the receiver clock based on OBSDATA_EPOCH, orbit

in this function, we solve position and clock together, but set positions at tight constraint

参数
obsdataan OBSDATA_EPOCH struct pointer
runtime_poda RUNTIME_POD_EMBED struct pointer
返回
0: ok , 1: no initial value obtained has to return, 2: use orbit integration as initial value , -1: kalman filter error, has to return
